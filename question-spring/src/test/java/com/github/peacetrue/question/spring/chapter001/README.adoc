== 总结
spring bean 相互依赖是没有问题，需要注意以下几点：

. 尽可能避免使用构造函数依赖
. 双向构造函数依赖肯定会导致循环引用异常
. 如果一方使用构造函数依赖，那么必须保证另一方先被初始化
